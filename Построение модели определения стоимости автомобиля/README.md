# Проект по теме "Численные методы"
Построение модели определения стоимости автомобиля

## Входные данные
Информация о машине (её параметры)

## Задача
Необходимо построить модель, которая будет определять стоимость автомобиля по её параметрам (важна как точность, так и время обучения и предсказаний)


## Используемые библиотеки
`pandas` `numpy` `matplotlib` `time` `sidetable` `inflection` `seaborn` `lightgbm` `sklearn`
